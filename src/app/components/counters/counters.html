Services & Dependency Injection

Share state across components using injectable services.

What is a Service?

A service holds reusable logic and state. Dependency Injection (DI) supplies instances where needed. Decorate classes with @Injectable() and provide them with providedIn: 'root' for a shared singleton across the entire app.

Shared Counter Service

Both Counter A and Counter B inject the same CounterService instance. Updating one reflects in the other.

Component-Provided Service (Hierarchical DI)

Provide a service in a component's providers array to create a local instance for its subtree. Sibling subtrees receive separate instances.
